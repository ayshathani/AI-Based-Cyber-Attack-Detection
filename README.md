# AI-Based-Cyber-Attack-Detection
Overview

This project presents an AI-based Intrusion Detection System (IDS) designed to detect Distributed Denial of Service (DDoS) attacks in a virtual smart city network.

Features
Built a virtual smart city environment
Captured network traffic using Wireshark
Applied machine learning techniques
Detected DDoS attacks
Analyzed malicious traffic
